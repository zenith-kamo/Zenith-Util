

<h1 align="center">Zenith Util</h1>

Zenith Util は、wplaceの利用体験をより快適に、そして自分好みにカスタマイズするためのスクリプトです。
洗練された設定画面から、複雑な操作なしでサイトの見た目や挙動を調整することができます。
---

## 主な機能

### プラグイン管理システム

すべての機能は独立した「プラグイン」として構成されています。必要な機能だけを選んで有効化できるため、動作を軽量に保つことが可能です。

### インターフェースのカラーカスタマイズ (Change Color)

サイト内の各種パーツの色を自由に変更できます。

* ボタンの背景色および文字色
* アクセントとなるプライマリカラー
* レベル表示などのサブカラー
* メニュー画面などの背景色（メイン・サブ）

### UIのクリーンアップ (No UI)

画面上の不要な要素を非表示にし、視認性を向上させます。

* 「Zoom in to see pixels」の非表示
* 画面端に表示される通知（トースト通知）の非表示

### 設定の保存と移行 (Import / Export)

作成したお気に入りの設定をJSON形式で保存できます。バックアップとして手元に残したり、別のブラウザやPCへ設定をそのまま移したりすることが可能です。

---

## 使用方法

### 1. インストール

お使いのブラウザに Tampermonkey等のスクリプトマネージャーがインストールされていることを確認し、スクリプトを登録してください。

### 2. 設定画面を開く

wplace.live を開くと、画面内の右側のボタン群に、設定アイコンが追加されます。このボタンをクリックすると管理画面が開きます。

### 3. 機能の有効化と詳細設定

* **有効化**: 各プラグイン名の横にあるスイッチを切り替えることで、機能を即座に反映できます。
* **詳細設定**: 各プラグインカード内にある小さな歯車アイコンをクリックすると、色選択やオプション項目の変更が可能です。

### 4. 設定のバックアップ

「Import/Export」タブから現在の設定コードをコピーして保存しておくことができます。
